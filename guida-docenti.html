A Cosa Serve Questa Guida

Questa guida accompagna il docente passo dopo passo nell'uso di AI School Template, dalla creazione delle lezioni alla loro condivisione con gli studenti.

  • creare lezioni digitali attraverso una procedura guidata
  • salvare le lezioni come file di backup
  • pubblicarle online in modo semplice e gratuito
  • condividere un link unico con i tuoi studenti

Il template è pensato per supportare il lavoro del docente, senza richiedere attività di programmazione o competenze da sviluppatore. Ogni scelta didattica resta sempre sotto il controllo del docente: il sistema non prende decisioni al posto tuo, ma ti accompagna nella progettazione dei contenuti.

Introduzione

AI School Template è uno strumento pensato per permettere ai docenti di progettare e pubblicare lezioni digitali in modo strutturato, accessibile e replicabile.

Il template ti permette di:

  • progettare lezioni organizzate in sezioni chiare
  • personalizzare contenuti e impostazioni grafiche
  • preparare materiali fruibili dagli studenti da qualsiasi dispositivo
  • pubblicare le lezioni online tramite un semplice link
Tutto il processo avviene attraverso passaggi guidati, senza attività di programmazione. L'uso dell'intelligenza artificiale, quando presente, è sempre facoltativo e dichiarato: il docente mantiene il controllo sui contenuti, sulle scelte didattiche e sui materiali condivisi con gli studenti.

Inizio Rapido

PASSO 1: Accedi al Template Docenti
  1. Il docente accede al Template Docenti tramite link.
  2. Non è necessario scaricare né installare file.
  3. Il lavoro di creazione e modifica delle lezioni avviene interamente online.
PASSO 2: Pronto per Creare
  1. Aprendo il link, il template si carica direttamente nel browser.
  2. Non serve installare nulla.
PASSO 3: Inizia a Creare
  1. Clicca "Dashboard Docente"
  2. Clicca "Crea Nuova Lezione"
  3. Compila i campi e salva!
  4. Puoi crearne quante ne vuoi prima di pubblicarle.
Hai creato la tua prima lezione.
IMPORTANTE – Backup delle lezioni
Le lezioni che crei vengono salvate nel browser che stai usando.
Se svuoti la cache del browser, cambi computer o browser senza aver esportato le lezioni, potresti perderle.
Cosa fare sempre:
  • Esporta spesso le lezioni come file di backup
  • Salva il file in una cartella sicura (PC, chiavetta o cloud)

Creare Lezioni

Procedura Guidata per Creare una Lezione

Il sistema ti guida attraverso 6 passaggi:

Step 1: Configurazione

  • Titolo: es. "Il Sistema Solare"
  • Sottotitolo: breve descrizione
  • Layout: scegli tra A-F (vedi anteprima)
  • Avatar: immagine rappresentativa
  • Tema: colori (Standard/Alto Contrasto/Bassa Luminosità)

Step 2: Contenuto Lezione

  • Testo della spiegazione
  • Video (URL YouTube)
  • Trascrizione (per accessibilità)
  • Registro IA (se hai usato intelligenza artificiale)

Step 3-6: Altre Sezioni

Attività, Risorse, Valutazione, Info & Etica IA

Suggerimento: Crea più lezioni prima di pubblicare. Puoi salvarle tutte insieme!

Esportare

Quando hai finito di creare tutte le lezioni che vuoi pubblicare:

Procedura di Esportazione
  1. Vai alla Dashboard Docente
  2. Clicca il pulsante "Esporta Tutte le Lezioni"
  3. Il browser chiede dove salvare il file: scegli liberamente la cartella.
  4. Il file esportato si chiama: lezioni-YYYY-MM-DD.json
IMPORTANTE: Questo file contiene TUTTE le tue lezioni. Conservalo come backup!

Dove Salvare il File

Quando esporti, scegli una cartella sicura come:

  • Chiavetta USB
  • Google Drive / OneDrive
  • Email a te stesso
  • Cartella dedicata sul PC
Suggerimento: Esporta spesso! Ogni volta che finisci di lavorare, fai un backup.

Pubblicare

Per pubblicare le lezioni online puoi usare Netlify Drop: non serve creare account né configurare nulla. Basta trascinare la cartella del template studenti.

PASSO 1: Prepara i File

  1. Vai alla Dashboard Docente
  2. Clicca "Esporta Tutte le Lezioni"
  3. Rinomina il file esportato da lezioni-YYYY-MM-DD.json a lessons.json
  4. Copia il file lessons.json nella cartella dedicata alle lezioni all'interno del template studenti

PASSO 2: Pubblica su Netlify

  1. Apri il browser e vai su: https://app.netlify.com/drop
  2. Trascina l'intera cartella del template studenti nella finestra
  3. Aspetta qualche secondo mentre carica
  4. Vedrai il link del tuo sito pubblicato

Il sito è online all'indirizzo:

https://nome-sito.netlify.app

Gli studenti accedono direttamente alle lezioni tramite:

https://nome-sito.netlify.app/student-home.html

Entrambi gli indirizzi funzionano. Il secondo è quello da condividere con gli studenti.

PASSO 3: Personalizza il Nome (Opzionale)

Se vuoi un nome più bello tipo lezioni-prof-rossi.netlify.app:

  1. Clicca "Claim this site" sulla pagina Netlify
  2. Crea account gratuito (email + password)
  3. Vai su "Site settings" → "Change site name"
  4. Scrivi il nome che vuoi (es: lezioni-prof-rossi)
  5. Salva!
Suggerimento: Salva il link del sito da qualche parte sicura! Serve per condividerlo con gli studenti e per aggiornarlo in futuro.

Aggiornare

Hai già pubblicato il sito e vuoi aggiungere o modificare lezioni? Segui la stessa procedura.

Procedura Completa:
  1. Nel Template Docenti, aggiungi o modifica le lezioni
  2. Dashboard → Clicca "Esporta Tutte le Lezioni"
  3. Rinomina il file in lessons.json
  4. Copia lessons.json nella cartella dedicata alle lezioni del template studenti
  5. Vai su https://app.netlify.com/drop
  6. Trascina di nuovo tutta la cartella del template studenti
  7. Il sito viene aggiornato automaticamente
IMPORTANTE: Ottieni un nuovo link ogni volta che ricarichi!
Per mantenere lo stesso link, devi usare l'account Netlify (opzione "Claim this site").

Mantenere lo Stesso Link

Se hai creato un account Netlify e "claimed" il sito:

  1. Fai login su netlify.com
  2. Vai sui tuoi "Sites"
  3. Clicca sul tuo sito (es: lezioni-prof-rossi)
  4. Clicca "Deploys" in alto
  5. Trascina la nuova cartella nel box "Drag and drop"
  6. Aspetta il deploy completato: stesso link, contenuti aggiornati

Recuperare Lezioni (Cambio PC/Browser)

  1. Accedi di nuovo al Template Docenti tramite il link
  2. Vai alla Dashboard
  3. Clicca "Importa Lezioni"
  4. Seleziona il file lessons.json salvato
  5. Tutte le lezioni vengono recuperate
Esporta sempre un backup dopo ogni modifica importante.

Problemi Comuni

"Non vedo le mie lezioni sul sito Netlify"

Controlli:

  • Il file lessons.json è nella cartella prevista per i contenuti del sito studenti?
  • Il file si chiama ESATTAMENTE lessons.json (no date, no maiuscole)?
  • Stai andando su https://...netlify.app/student-home.html?
  • Hai trascinato la cartella COMPLETA su Netlify?

"Ho perso le mie lezioni!"

Soluzioni:

  • Hai il file lessons.json salvato? → Usa "Importa Lezioni"
  • Controlla le cartelle dove hai salvato i file di backup
  • Controlla backup su chiavetta USB o cloud

"Il link cambia ogni volta"

Problema: Netlify Drop crea un nuovo link random ogni volta.

Soluzione:

  • Crea un account Netlify gratuito
  • Clicca "Claim this site" dopo il primo upload
  • Personalizza il nome (es: lezioni-prof-rossi.netlify.app)
  • Usa sempre il pannello Netlify per aggiornare → stesso link!

"Il sito è lento"

Se il sito è lento:

  • Controlla la tua connessione internet
  • Prova a ricaricare la pagina (F5)
  • Evita di caricare video/immagini troppo pesanti

Informazioni sul progetto

AI School Template è un progetto indipendente pensato per supportare i docenti nella progettazione di lezioni digitali accessibili, inclusive e replicabili. Il template nasce dall'esperienza didattica sul campo e dall'esigenza di offrire uno strumento semplice e trasparente.

Il template guida il docente nell'inserimento dei contenuti passo dopo passo, favorendo un controllo umano consapevole su ogni fase del processo di costruzione della lezione, in particolare quando è previsto il supporto dell'intelligenza artificiale.

L'uso dell'intelligenza artificiale è concepito come supporto al lavoro docente, nel rispetto dei principi di controllo umano, responsabilità, accessibilità e tutela dei dati.
